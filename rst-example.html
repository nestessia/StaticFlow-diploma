============================
Пример формата reStructuredText
============================

reStructuredText (RST) - это легкий язык разметки, используемый преимущественно в документации Python-проектов.

Основные возможности

System Message: WARNING/2 (<string>, line 8)

Title underline too short.

Основные возможности
===================

Заголовки

В RST заголовки создаются с помощью подчеркивания (и возможно надчеркивания). Уровень заголовка определяется последовательностью использования разных символов подчеркивания.

Форматирование текста

  • Жирный текст используется с двойными звездочками

  • Курсив обозначается одинарными звездочками

  • Моноширинный текст обозначается двойными обратными кавычками

Списки

Маркированные списки:

  • Первый пункт

  • Второй пункт * Вложенный пункт * Еще один вложенный пункт

  • Третий пункт

Нумерованные списки:

  1. Первый пункт

  2. Второй пункт a. Вложенный A b. Вложенный B

  3. Третий пункт

Код

Встроенный код: print("Hello,World!")

Блок кода с подсветкой синтаксиса:

defhello_world():"""Функция для приветствия мира."""print("Hello, World!")ifTrue:print("Истина!")# Циклforiinrange(3):print(f"Итерация {i}")hello_world()
