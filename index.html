Добро пожаловать в StaticFlow

StaticFlow - это современный генератор статических сайтов с богатыми возможностями для создания контента.

Возможности

1. Подсветка кода

def hello_world():
    print("Привет, StaticFlow!")

    if True:
        print("Tabs are working!")
        for i in range(3):
            print(f"Tab level {i+1}")

Here's some JavaScript with tabs:

function testFunction() {
    console.log("Testing tabs");

    if (true) {
        console.log("Nested level");

        for (let i = 0; i < 3; i++) {
            console.log(`Loop iteration ${i}`);
        }
    }
}

5. Разные форматы контента

StaticFlow поддерживает различные форматы для создания контента:

  • Markdown - основной формат (который вы сейчас читаете)
  • reStructuredText - популярный в мире Python
  • AsciiDoc - мощный формат для технической документации

Перейдите по ссылкам, чтобы увидеть примеры каждого формата.
